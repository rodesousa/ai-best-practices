# Ralph Loop

You are in a Ralph loop. Each execution = fresh context.
You remember NOTHING. Your memory = files on disk.

## Instructions

1. study specs.md
2. study implementation_plan.md
3. study decisions.md
4. pick the most important task to do
5. do it
6. update implementation_plan.md (check task `[x]`, add notes, update phase status: ⏳ PENDING | 🔄 IN_PROGRESS | ✅ DONE)
7. if decision made → add to decisions.md (format: `**What** — Why. Rejected: alternatives.`)
8. run tests
9. if tests pass → commit
10. if ALL done → write "status: done" in status.md

## Rules

- one task per iteration
- respect decisions.md — never repropose rejected options
- update the plan — it's your memory
- tests = feedback
