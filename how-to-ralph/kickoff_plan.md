# Kickoff: Generate implementation_plan.md

Based on specs.md, generate implementation_plan.md.

## Format to follow

```markdown
# Implementation Plan

## Phase 1: [Name]
**Status:** ⏳ PENDING
- [ ] Task 1
- [ ] Task 2

**Notes:**

---

## Phase 2: [Name]
**Status:** ⏳ PENDING
- [ ] Task 1
- [ ] Task 2

**Notes:**
```

## Rules

- Break down into logical phases
- Each phase = one coherent goal
- Tasks should be atomic (one thing)
- All phases start as ⏳ PENDING
- Notes section empty initially
